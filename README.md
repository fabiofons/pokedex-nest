# Run project in development env

1. Clone the repository

2. Run 
```pnpm install```

3. Had Nest CLI installed
```pnpm i -g @nestjs/cli```

4. Setup the DB
```docker-compose up -d```


## Stack
* MongoDB
* NestJS
